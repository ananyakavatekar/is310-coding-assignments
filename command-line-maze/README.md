# The Invisibility of Womens' Labor
**Author:** Ananya Kavatekar

Recommended commands:
- `ls` to list files and folders
- `cd` to move between directories
- `cat` to read text files
- `ls -la` to reveal hidden files

## Instructions
1. Begin in the `entrance` directory.
2. Read files as you move through the folder structure.
3. Pay attention to how different types of labor are recorded.
4. The maze is solved when you reach the file in the `solution` directory.
